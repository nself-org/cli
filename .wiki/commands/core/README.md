# Core Commands

The 5 essential commands for working with nself.

## Commands

- [init](init.md) - Initialize a new nself project
- [build](build.md) - Generate configuration files
- [start](start.md) - Start all services
- [stop](stop.md) - Stop all services
- [restart](restart.md) - Restart services

## Overview

These are the core commands you'll use in every nself workflow:

```bash
# Initialize project
nself init

# Generate configs
nself build

# Start services
nself start

# Stop services
nself stop

# Restart after changes
nself restart
```
