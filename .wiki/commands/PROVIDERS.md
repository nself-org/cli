# nself providers - Cloud Providers (Legacy)

**Version 0.9.9** | Manage cloud providers

---

## Overview

The `nself providers` command manages cloud provider configurations. This is a legacy alias for `nself provider`.

---

## Usage

```bash
# List providers
nself providers list

# Equivalent to:
nself provider list
```

---

## Supported Providers

See [Complete Provider List](../guides/PROVIDERS-COMPLETE.md) for all 26 supported cloud providers.

---

## Note

For new projects, use `nself provider` instead. The `providers` command is maintained for backward compatibility.
