# Installation

Install nself on your system.

## macOS

```bash
brew tap acamarata/nself
brew install nself
```

Or use npm:
```bash
npm install -g @nself/cli
```

## Linux

```bash
curl -sSL https://install.nself.org | bash
```

Or use npm:
```bash
npm install -g @nself/cli
```

## Windows (WSL)

Use Windows Subsystem for Linux and follow Linux instructions.

## From Source

```bash
git clone https://github.com/acamarata/nself.git
cd nself
./install.sh
```

## Verify Installation

```bash
nself version
nself help
```
