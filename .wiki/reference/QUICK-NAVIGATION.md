## Common Workflows

### Local Development
```bash
# See: Quick Start Guide
nself init
nself build
nself start
nself db schema apply schema.dbml
```
**[Quick Start Guide](../getting-started/Quick-Start.md)**

---

### Database Development
```bash
# See: Database Workflow
nself db schema scaffold saas
# Edit schema.dbml
nself db schema apply schema.dbml
nself db types generate
```
**[Database Workflow](../guides/DATABASE-WORKFLOW.md)**

---

### Production Deployment
```bash
# See: Deployment Guide
nself env create prod production
# Edit server.json
nself deploy prod
nself prod status
```
**[Deployment Guide](../deployment/README.md)**

---

### Plugin Installation
```bash
# See: Plugin Guide
nself plugin list
nself plugin install stripe
# Configure .env
nself build && nself restart
nself plugin stripe sync
```
**[Plugin Overview](../plugins/README.md)**
